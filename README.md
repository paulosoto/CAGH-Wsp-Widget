# GH WhatsApp Widget

Floating WhatsApp widget with animated QR popup for Clínica Adventista Good Hope.

## Installation

```html
<script
  src="src/gh-whatsapp-widget.js"
  data-phone="51944648831"
  data-mode="auto"
  data-position="right"
  data-color="#25D366"
  data-message="Habla con nosotros 💬"
></script>
```
